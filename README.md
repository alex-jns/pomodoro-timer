# Focus Flow Pomodoro

A static website created solely by OpenAI's Codex as a proof of concept. Features a pomodoro timer to assist in studying, work, or focusing on tasks. Supports YouTube links to play music, removing the need for extra tabs that could distract you.

Check it out here: [Focus Flow Pomodoro](https://alex-jns.github.io/pomodoro-timer/)

# Features

- Pomodoro timer with varying session modes
- Subtle alarm upon session completion
- Trackers that count sessions and time elapsed
- Describe the intent of your next session to stay focused
- Ability to toggle to a dark mode theme that your browser remembers
- Soundtrack section that supports YouTube links
  - Defaults to the "lofi hip hop radio" by [Lofi Girl](https://www.youtube.com/channel/UCSJ4gkVC6NrvII8umztf0Ow)

# Usage

Option 1 - Open the link in your browser:

```
https://github.com/alex-jns/pomodoro-timer
```

Option 2 - Clone or download the repository:

```
git clone https://github.com/alex-jns/pomodoro-timer.git
```

Note: The YouTube links will not play locally due to protocol restrictions. It is recommended to run a local server to properly use this feature.
